# 관리자 콘솔 QA 체크리스트

## 1. 로그인/토큰
- [ ] 관리자 계정 로그인 시 accessToken이 입력창에 채워진다.
- [ ] 토큰 저장/삭제가 정상 동작한다.
- [ ] 잘못된 토큰으로 관리자 API 호출 시 에러가 표시된다.

## 2. 반 조회/필터
- [ ] `GET /api/admin/year-classes` 결과가 카드로 표시된다.
- [ ] 반명/교사명/학생명/활성 필터가 정상 동작한다.
- [ ] 카드의 `이 반을 작업 대상으로 설정` 클릭 시 배정 입력값이 자동 채워진다.

## 3. 반 수정(인라인)
- [ ] 카드에서 반명/정렬/활성 수정 후 저장된다.
- [ ] 잘못된 값(빈 반명, 잘못된 정렬값)에 대한 에러가 표시된다.

## 4. 교사 배정/해제
- [ ] 폼 배정/해제 요청 전에 confirm이 뜬다.
- [ ] 카드 내 개별 해제 버튼이 정상 동작한다.
- [ ] 카드 내 일괄 해제가 정상 동작한다.
- [ ] 마지막 작업 되돌리기(Undo)로 직전 작업이 역연산된다.

## 5. 학생 배정/해제/이동
- [ ] 폼 배정/해제/이동 요청 전에 confirm이 뜬다.
- [ ] 카드 내 개별/일괄 해제가 정상 동작한다.
- [ ] 이동 후 대상 반에서 학생이 보이고, Undo 시 원래 반으로 복구된다.
- [ ] 다른 반 중복 규칙(한 연도 한 반)이 정상적으로 강제된다.

## 6. 교사/학생 풀
- [ ] 서버사이드 페이징으로 조회된다(limit/offset 반영).
- [ ] 이전/다음 페이지 버튼으로 페이지 이동된다.
- [ ] 페이지가 바뀌어도 선택된 체크 상태가 유지된다.
- [ ] 다중 선택 배정이 정상 동작한다.

## 7. 운영 로그
- [ ] 로그 조회(limit/offset/actionType/keyword/fromAt/toAt) 동작 확인.
- [ ] actionType 목록 로딩 버튼이 정상 동작한다.
- [ ] 기간 preset(오늘/7일/30일) 버튼이 정상 입력한다.
- [ ] CSV 다운로드가 정상 동작한다.

## 8. 운영 배포 안전장치
- [ ] `app.admin-console.enabled=true` 에서 `/admin.html` 접근 가능.
- [ ] `app.admin-console.enabled=false` 에서 `/admin.html`, `/admin.js`, `/admin.css`가 404 응답.
